# مرفأ للاستشارات — Marfa Consulting

موقع إلكتروني احترافي وعصري لشركة استشارات أعمال متخصصة في تقديم استشارات مباشرة وفعالة للشركات الكبرى والمتوسطة.

## 📋 نظرة عامة

**مرفأ** هي شركة استشارات أعمال متخصصة في مساعدة الشركات على اتخاذ قرارات أوضح وأسرع وأقل تكلفة. الموقع الإلكتروني يعكس هذه القيم من خلال تصميم احترافي وتجربة مستخدم ممتازة.

### المميزات الرئيسية

✨ **تصميم احترافي عصري**
- تصميم متجاوب يعمل على جميع الأجهزة
- نظام ألوان احترافي وعالي التباين
- حركات وانتقالات سلسة وجذابة

🎯 **تجربة مستخدم ممتازة**
- واجهة بديهية وسهلة الاستخدام
- ملاحة سلسة وسريعة
- نموذج اتصال محسّن مع التحقق من الصحة

⚡ **أداء عالي**
- تحميل سريع وتحسينات SEO
- كود محسّن وقابل للصيانة
- دعم كامل للأجهزة المحمولة

♿ **إمكانية الوصول**
- دعم كامل للملاحة بلوحة المفاتيح
- علامات ARIA مناسبة
- دعم قارئات الشاشة

## 🚀 البدء السريع

### المتطلبات

- متصفح حديث (Chrome, Firefox, Safari, Edge)
- خادم ويب بسيط (اختياري)

### التثبيت

1. استنساخ المستودع:

```bash
git clone https://github.com/yourusername/marfa-consulting.git
cd marfa-consulting
```

2. فتح الملف مباشرة:

```bash
# على macOS
open index.html

# على Linux
xdg-open index.html

# على Windows
start index.html
```

3. أو استخدام خادم ويب محلي:

```bash
# باستخدام Python 3
python -m http.server 8000

# ثم افتح المتصفح على:
# http://localhost:8000
```

## 📱 التصميم المتجاوب

الموقع مُحسّن للعمل على:

- **الأجهزة الكبيرة**: 1920px وأكثر
- **الأجهزة المتوسطة**: 880px - 1920px
- **الأجهزة الصغيرة**: 480px - 880px
- **الهواتف الذكية**: أقل من 480px

## 🐛 استكشاف الأخطاء

### الموقع لا يحمل

- تحقق من أن جميع الملفات موجودة
- تحقق من أذونات الملفات
- استخدم أدوات المطور (F12) للتحقق من الأخطاء

### النموذج لا يعمل

- تحقق من أن JavaScript مفعّل
- تحقق من عنوان البريد الإلكتروني
- تحقق من وحدة التحكم (Console) للأخطاء

### الموقع بطيء

- امسح ذاكرة التخزين المؤقت
- تحقق من سرعة الإنترنت
- استخدم أدوات تحليل الأداء
